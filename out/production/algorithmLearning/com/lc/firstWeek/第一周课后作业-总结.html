1.删除排序数组中的重复项
public int removeDuplicates(int[] nums) {
        if (nums == null || nums.length ==0){
            return 0;
        }
        int numsSize = nums.length;
        int i = 0;
        int j = 1;
        for (j = 1; j <numsSize; j++) {
            if (nums[j] != nums[i]){
                i++;
                nums[i]=nums[j];
            }
        }
        return i + 1;
    }
题意：对排好序的数组进行去重，并返回数组中剩余元素的个数，且要求只能对原数组进行操作，不可额外创建数组
思路：通过快慢指针 快指针按顺序指向数组中的下标，慢指针指向非重复数据在数组中应该存储的位置的下标，将快指针每次指向的元素与慢指针指向的元素进行对比，只有当元素不同时慢指针指针偏移一位同时赋予指针位置的值为新出现的值，循环完整个数组即可去除所有重复值，且从左向右填充数组，慢指针的值为数组中最后一个值的下标，对应的数组中元素个数就是指针数字+1；
总结：对于数组中的元素进行有规则的两两比较时可以使用快慢指针的方式

2.旋转数组
    public void rotate(int[] nums, int k) {
        if (nums == null || nums.length == 0 || nums.length == 1) {
            return;
        }
        int numsSize = nums.length;
        if (k == 0) {
            return;
        }
        int[] tmpNums = new int[numsSize];
        for(int i = 0;i < numsSize;i++){
            if((i+k) < numsSize){
                tmpNums[i+k]=nums[i];
            }else{
                tmpNums[(i+k)%numsSize] = nums[i];
            }
        }
        for(int j=0;j<numsSize;j++){
            nums[j] = tmpNums[j];
        }
        
    }
题意：给定一个数组及数字k根据k值将数组中元素进行向右旋转
思路：根据原数组中元素的位置，以及旋转k个位置后的元素所在位置可以总结发现
         if((i+k) < numsSize){
                tmpNums[i+k]=nums[i];
            }else{
                tmpNums[(i+k)%numsSize] = nums[i];
            }
            因此可以利用中间数组 获取正确位置的的元素，再次赋值给给定数组
总结：适当使用中间变量可以让问题简单化
3.合并两个有序链表
           public ListNode mergeTwoLists(ListNode l1, ListNode l2) {
            if(l1 == null){
                return l2;
            }
            if(l2 == null){
                return l1;
            }
            if(l1.val < l2.val){
                l1.next = mergeTwoLists(l1.next,l2);
                return l1;
            }else{
                l2.next = mergeTwoLists(l2.next,l1);
                return l2;
            }
            return null;
        }                                                                                   

题意：将两个有序链表合并为一个新的有序链表
思路：两个有序链表假设存在一个为空则返回另一个就ok,假设不为空判断两个链表当前值的大小并通过递归调用直到链表有一个为空时返回另外一链表
总结：递归三要素
               1）确定递归函数做哪些处理
                              2）要有结束条件
                              3）等价关系
